Frequently Asked Questions

What file formats are supported? +

ModelSwap supports all major 3D model formats including OBJ, STL, PLY, COLLADA (.dae), glTF (.gltf), GLB (.glb), and OFF (.off). You can convert between any of these formats.

What is the maximum file size? +

The maximum file size is 100MB per file. For larger files, consider splitting them into smaller parts or using specialized 3D software.

How long does conversion take? +

Conversion time depends on file size and complexity. Small files (under 10MB) typically convert in a few seconds, while larger files may take up to a minute. The process is optimized for speed and efficiency.

Are my files safe and private? +

Yes! Your files are processed locally and are automatically deleted after conversion. We don't store, share, or access your files. All processing happens on your device for maximum privacy and security.

Can I convert multiple files at once? +

Currently, we support single file conversions to ensure optimal performance and prevent memory issues. Please convert files one at a time for the best experience.

What if my conversion fails? +

If conversion fails, check that your file is a valid 3D model in a supported format. Common issues include corrupted files, unsupported formats, or files that are too large. The application will show specific error messages to help you troubleshoot.

Do you support textures and materials? +

Yes, we support textures and materials where the format allows it. Formats like OBJ, COLLADA, and glTF preserve texture information. However, some formats like STL are geometry-only and will not preserve texture data.

Is this application free to use? +

Yes! ModelSwap is completely free and open source. You can use it for personal or commercial projects without any restrictions. The source code is available on GitHub for transparency and community contributions.
